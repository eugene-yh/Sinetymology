# 保释
## Meta
* traditional Chinese: 保釋
* pronunciation: bǎoshì
* meaning: 
   - [en] release upon bail (Masini, 1993)
   - [zh] 被羁押的犯罪嫌疑人、被告人根据法律的规定取保获释。(现代汉语词典第五版)
* structure: determining-determined

## Academic hypotheses
### Claims

2. [Gao and Liu, 1958]
